# SSE - Server-Sent Events

Streaming data from server to client using Server-Sent Events (SSE) in a web application. One-way communication from
server to client.

* Service to service -> APPLICATION_NDJSON_VALUE
* Service to browser -> TEXT_EVENT_STREAM_VALUE

## Use Cases

* Real-time notifications
* Live updates (e.g., news feeds, stock prices)
* Monitoring dashboards
* Chat applications (one-way updates)
